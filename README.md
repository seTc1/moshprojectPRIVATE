# Анализ поступления — десктопное приложение на Python

Это учебный проект, реализующий небольшую систему для анализа конкурсных
списков абитуриентов. Приложение построено по принципам объектно‑
ориентированного программирования, использует SQLite для хранения данных
и PyQt для построения графического интерфейса. Помимо загрузки и
просмотра списков, система умеет рассчитывать проходные баллы с учётом
приоритетов заявлений, строить простые отчёты в формате PDF и
генерировать статистику.

## Возможности

- Хранение данных об образовательных программах, абитуриентах и их
  заявлениях в локальной базе SQLite.
- Загрузка конкурсных списков из CSV‑файлов и обновление существующих
  записей.
- Просмотр списков в удобной таблице с возможностью сортировки и
  фильтрации по программе и дате.
- Рассчёт проходного балла по каждому направлению с учётом
  приоритетов (каскадный алгоритм).
- Формирование отчёта в PDF с динамикой проходных баллов, списками
  зачисленных и статистической таблицей.
- Генератор тестовых данных, позволяющий быстро заполнить базу
  случайными списками для демонстрации.

## Требования

- Python 3.8+
- Библиотека `pandas`
- Библиотека `matplotlib`
- Библиотека `fpdf2`
- Qt‑библиотека: `PyQt5` или `PyQt6` (одно из двух). В среде, где
  отсутствует поддержка Qt, графический интерфейс запустить не удастся,
  однако логические компоненты и отчёты продолжат работать.

## Установка

1. Клонируйте репозиторий или скопируйте исходный код в рабочую папку.
2. Создайте виртуальное окружение (рекомендуется) и установите
   зависимости:
   ```bash
   pip install pandas matplotlib fpdf2 PyQt5
   ```
   При отсутствии `PyQt5` можно установить `PyQt6` или `PySide6`.

## Запуск

```bash
python -m admission_project.main
```

При первом запуске программа создаст файл `admission.db`, заполнит его
тестовыми данными и откроет графический интерфейс. В дальнейшем можно
загружать собственные списки из CSV — для этого выберите программу,
дату и нажмите «Load CSV…».

## Тесты

Для проверки корректности логики обновления списков и расчёта
проходных баллов предусмотрены юнит‑тесты на базе `pytest`. Запустить
их можно командой:

```bash
pytest admission_project/tests
```

## Структура проекта

```
admission_project/
├── database.py       — слой работы с базой данных (SQLite)
├── report.py         — генерация отчётов в формате PDF
├── gui.py            — графический интерфейс на PyQt5/PyQt6
├── data_generator.py — генерация случайных данных для тестов и демо
├── main.py           — точка входа, инициализация БД, запуск GUI
└── tests/
    └── test_database.py — несколько тестов для основных операций
```

## Замечания

- Проект предназначен для демонстрационных целей и не реализует все
  требования из кейса (например, строгие пересечения между программами
  при генерации списков). Основные идеи, такие как каскадное
  распределение мест, обновление списков и формирование отчётов,
  представлены.
- В рамках ограниченного окружения PyQt может быть недоступен — в этом
  случае доступна только логическая часть проекта без GUI.
